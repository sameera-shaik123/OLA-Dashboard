# 🚖 OLA Ride Cancellation Analysis  
### Identifying Trends & Reasons for Ride Cancellations

---

## 1️⃣ 📌 Project Title / Headline

**OLA – Identified the Trends and Reasons for Ride Cancellation Using SQL & Power BI**

---

## 2️⃣ 📖 Short Description / Purpose

This project focuses on analyzing OLA ride booking data to identify patterns, trends, and key reasons behind ride cancellations.

The objective was to:
- Understand cancellation behavior
- Compare driver vs customer cancellations
- Calculate cancellation rate
- Identify operational gaps
- Generate business insights for reducing ride cancellations

The project involved cleaning and processing raw booking data from multiple sources, performing SQL-based analysis, and building an interactive Power BI dashboard to visualize trends.

---

## 3️⃣ 🛠 Tech Stack

- **SQL (MySQL Workbench)** → Data querying & analysis  
- **Microsoft Power BI** → Dashboard creation & visualization  
- **DAX (Data Analysis Expressions)** → KPI & measure calculations  
- **Excel / CSV** → Data source & preprocessing  

---

## 4️⃣ 📂 Data Source

- Structured OLA booking dataset (July data)
- Data included:
  - Booking ID
  - Booking Status
  - Cancellation Type
  - Date
  - Driver Details
  - Customer Details
  - Ride Information

(Data used for learning and analytical purposes.)
